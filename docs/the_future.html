Machine Learning

So, what is the future of this project !

This analysis is very relevant today, a user can trust it and make a decision.


But, what about 6 months from now, or a year or 2 years from now? Will this analysis be as useful as it is today?


Perhaps not!
By that time, there would have been tens of thousands of more visits to these hotels...
Perhaps new hotels would started services...
Perhaps some hotels from our list would have improved their services by a big margin, and others wouldn't...

Perhaps the perception of a Hotel in the minds of customers has changed by a small extent - or a large?


So, to make the analysis really relevant at that time in the future, we would have to incorporate all reviews at the time of viewing the results (and not just what we have today)

Here is what we plan to do

  1. We will capture the every review.
  2. We will used our saved Deep Learning NLP model to predict the sentiment of the review.
  3. We will add the data above to our PostgreSQL Database on RDS
  4. We will use Tableau to analyze the data

The data upload and analysis can be scheduled once a week/month.

Or it can happen real-time on the cloud.

POC (review capture and sentiment prediction)

  1. Open Google Colab
  2. Sign in with your Google credentials
  3. Upload this file on Google Colab
  4. Place this file at the root of the Colab file above
  5. Run the File: Runtime -> Run All
  6. Enter as many reviews as you want to test
  7. View the model predictions.
